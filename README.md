# Group Policy Troubleshooting Toolkit

A read-only PowerShell toolkit for Group Policy troubleshooting and escalation evidence.

## Features

- GPResult HTML and text export
- Group Policy event log summary
- Computer and user context
- Applied policy evidence folder
- CSV, TXT, and HTML outputs

## How to run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Group_Policy_Troubleshooting_Toolkit.ps1
```

## Safety

Diagnostic-only. It does not change Group Policy settings.
